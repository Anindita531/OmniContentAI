# 🚀 OmniContent AI

## 💠 Overview

OmniContent AI is an **AI-powered multi-agent system** that automates the entire lifecycle of enterprise content:

* ✍️ Content Creation
* 🛡️ Compliance Review
* 🌐 Localization
* 📢 Multi-channel Distribution

It significantly reduces content turnaround time while ensuring consistency and brand compliance.

---

## 🧠 Features

### 🤖 Multi-Agent Pipeline

* Writer Agent → Generates content
* Compliance Agent → Ensures brand/legal standards
* Localization Agent → Translates into multiple languages
* Distribution Agent → Formats for channels

### 🎯 Key Capabilities

* AI-generated marketing content
* Real-time compliance validation
* Multi-language output
* Channel-specific formatting (LinkedIn, Twitter, Email)
* Agent logs visualization
* Typing animation UI
* Copy-to-clipboard buttons
* Export to PDF

---

## 💎 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **AI Framework:** LangChain / LangGraph
* **LLM APIs:** OpenAI / Groq
* **PDF Export:** ReportLab
* **Deployment:** Render

## ⚙️ Installation

### 1. Clone Repo

```
git clone https://github.com/yourusername/yourrepo.git
cd yourrepo
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Add Environment Variables

Create `.env` file:

```
OPENAI_API_KEY=your_key
GROQ_API_KEY=your_key
```

---

## ▶️ Run Locally

```
streamlit run app.py
```

## 🌍 Deployment (Render)

### Build Command

## 🧩 How It Works

1. User inputs campaign brief
2. Writer Agent generates draft
3. Compliance Agent validates content
4. Localization Agent translates
5. Distribution Agent formats outputs
6. Final assets displayed in dashboard
